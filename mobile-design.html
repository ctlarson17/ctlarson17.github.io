Mobile Logs

This project was adding a feature to an app that is in production. It was identified that a need of users was to be able to see device logs in-app. This app was used during large events, and if there were issues that prevented the app from performing it would pose a serious risk to the event. The solution was to add logs to the app, and let users view these logs in real time. Users could then identify what the root cause of an issue to more quickly fix the problem.
